
Web Page is created with some example content and a database. First database should be imported.
To change it's content, admin need to login to system by going to "president.php" page.
For the first use, username equals "user", password equal "password".
 When signed in on "president.php" page, forms that sent by user can be seen, announcements and news can be seen. 
By filling the input fields against the headings and texts and then pressing the "Kaydet" button
they can be editted.
